一、实验简介

本实验使用 Arduino millis() 非阻塞计时函数配合状态机，实现 LED 循环输出标准 SOS 摩尔斯信号：

信号规则：3 次短亮 → 3 次长亮 → 3 次短亮，一轮完整 SOS 结束后保持长时间熄灭停顿，循环往复。

全程不使用 delay()，程序不会阻塞，可并行扩展其他任务。

二、millis () 核心知识点

函数作用

millis() 开机后从零持续计时，返回 unsigned long 类型毫秒时间戳，独立运行不阻塞主循环。

非阻塞优势

delay() 会卡死整个程序，等待期间无法处理按键、串口、传感器；

millis() 仅读取时间，循环持续运行，多任务可同时执行。

变量规范

时间记录变量必须使用 unsigned long，避免 int 类型数值溢出导致计时错乱。

定时逻辑

记录上一次动作时间 prevTime，每次循环读取当前时间 now，通过时间差判断是否到达定时节点，到达后更新时间戳、执行动作。

三、代码整体思路

参数定义

统一配置短亮、短灭、长亮、字符间隔、一轮结束大间隔，修改数值即可调整闪烁节奏。

状态机划分

将一轮 SOS 拆分为 6 个阶段：

S\_dot（第一段 3 短闪）→ gap1（间隔）→ O\_line（3 长闪）→ gap2（间隔）→ S\_dot2（第二段 3 短闪）→ endWait（一轮结束长停顿）

完成长停顿后自动切回开头，无限循环。

计时动态分配

根据当前阶段与 LED 亮灭状态，自动匹配对应等待时长：

LED 点亮时：短闪阶段使用短亮时长，长闪阶段使用长亮时长

LED 熄灭时：普通间隔使用短灭时长，一轮结束使用超长停顿

计数控制闪烁次数

变量 count 记录单次阶段内闪烁次数，达到 3 次自动切换下一阶段，实现 3 短 / 3 长固定摩尔斯格式。

串口调试输出

一轮 SOS 完整播放完毕打印提示文字，方便调试观测循环周期。

四、运行效果说明

上电后 LED 自动循环输出 SOS：

快速闪 3 次 → 短暂熄灭 → 慢速长亮 3 次 → 短暂熄灭 → 快速闪 3 次 → 长时间熄灭停顿；

每次完整一轮结束串口打印提示信息；

全程无阻塞，可在 loop 中新增按键、传感器等其他功能代码同步运行；

修改顶部 flashOn、longFlash、loopGap 常量，可自由调整闪烁快慢与停顿长度。
